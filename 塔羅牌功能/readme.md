隨機排序
() => Math.random() - 0.5 - 隨機比較函數

Math.random()會產出 0~1 之間隨機數字

// 正常排序（數字由小到大）
[3, 1, 2].sort((a, b) => a - b)  
// 結果：[1, 2, 3]

// 反向排序（數字由大到小）
[1, 2, 3].sort((a, b) => b - a)  
// 結果：[3, 2, 1]

// sort() 的比較邏輯：
比較函數回傳負數 → a 排在 b 前面
比較函數回傳正數 → b 排在 a 前面  
比較函數回傳 0 → 順序不變

array.sort(() => Math.random() - 0.5)
